# CFDNSClient
Tool that manages Cloudflare DNS A and AAAA records<br/>
This tool was made quickly as a supplement for DDClient, so the code is a bit janky

# How to use
Install [Deno](https://deno.land/)

Run either with `deno task dev` (watches for file edits) or `deno task run` (runs once)
> [!NOTE]
> Any extra arguments are passed to the app

You can build with `build:*` commands, builds are placed in the `bin` folder, are self contained and can run without Deno:
- Windows x64: `build:win`
- Linux x64: `build:linux`
- Linux arm64: `build:linux_arm`

# How it works
It uses Cloudflare's [Account API tokens](https://dash.cloudflare.com/?to=/:account/api-tokens) and [icanhazip.com](https://icanhazip.com) in order to update specific records from configured zones

Example config: (generated by running with `--write-config`)
```json
{
    "apiKey": "",
    "interval": 300000,
    "zones": [
        {
            "name": "example.com",
            "enableAAAA": true,
            "records": [
                "@",
                "www",
                "api"
            ]
        },
        {
            "name": "example.net",
            "enableAAAA": false,
            "records": [
                "@",
                "www"
            ]
        }
    ]
}
```
